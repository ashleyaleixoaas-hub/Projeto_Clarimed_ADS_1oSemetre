# Template padrão da Aplicação

O layout padrão do site foi construído com as linguagens de marcação HTML e CSS, e contém algums funcionalidades desenvolvidas em JavaSccipt.

Todas as paginas possuem um Header, body e footer. A pagina contem os seguintes itens de padronização:

Cores: #bdddf, #333, #5adbe4, #0350af98.

Font-Family: Bree Serif, serif.

O código utilizado para construção dos elementos citados, pode ser consultado [aqui](https://icei-puc-minas-pmv-ads.github.io/pmv-ads-2026-1-e1-Clarimed/codigo-fonte/home/home.html). As imagens e icones ultilizados [aqui](https://icei-puc-minas-pmv-ads.github.io/pmv-ads-2026-1-e1-Clarimed/codigo-fonte/home/home.html).
