# Gestão de medicamentos 

`Análise e Desenvolvimento de Sistemas`

`Desenvolvimento de aplicação web front-end`

`1º SEMESTRE`

A bula é uma das principais fontes de informação para pacientes que desejam compreender
melhor os medicamentos que utilizam. No entanto, a linguagem técnica presente nesses
documentos pode dificultar sua interpretação.
A utilização de medicamentos sem o devido acompanhamento médico pode resultar em
complicações graves de saúde. Como exemplo, recentemente, o número de casos envolvendo o
medicamento tirzepatida (Mounjaro) vem aumentando, o que levantou discussões sobre
os riscos associados ao uso inadequado de tratamentos e à falta de compreensão sobre
seus efeitos e possíveis interações.
